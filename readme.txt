PROJECT 3: Independent Project
--REFLECTION--
This is a proof of concept for an audio player widget that can be used on hobbyist personal websites through platforms such as Neocities. The only dependencies the project uses are Google Fonts and Font Awesome, this is to make it easy for a beginner to drop the widget into their own project. Aside from the cover images and Font Awesome icons, the entire interface was made in CSS, I'm especially happy with how the ring of buttons turned out. In future updates, I'd like to make the code more user friendly, and include a tutorial for how to add the widget to your website in the index.html page.
